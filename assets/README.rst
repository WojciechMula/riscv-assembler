================================================================================
                Obtaining input for the generator
================================================================================

Please run `./prepare.sh` (or see what's there).

Prerequirements:

* checked out https://github.com/riscv/sail-riscv/
* run `./build_simulators.sh` in the `sail-riscv` project to create
  configuration files
* installed `sail` binary, it should be available in your path
